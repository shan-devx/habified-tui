# habified-tui
A terminal-based habit tracker and break timer built in C++ 20

[habified-tui.webm](https://github.com/user-attachments/assets/9287e91c-eb9d-4b56-b968-f6bc6f4411d7)

# Dependencies
- Cmake (v3.16+)
- C++20 compatible compiler (g++)
- [FTXUI](https://github.com/ArthurSonzogni/FTXUI)
- [nlohmann/json](https://github.com/nlohmann/json)

# How to Build and Run
1. **Clone the repository**
```bash
git clone https://github.com/shan-devx/habified-tui
cd habified-tui
```
2. **Build the project using CMake**
```bash
mkdir build
cd build
cmake ..
make
```
3. **Run the app**
```bash
./habified-tui
```
